# Греческий алфавит

| Символ                 | LaTeX            | Обозначает |
| ---------------------- | ---------------- | ---------- |
| $\Xi, \xi$             | \Xi, \xi         | Кси        |
| $\Sigma, \sigma$       | \Sigma, \sigma   | Сигма      |
| $\Pi$, $\pi$, $\varpi$ | \Pi, \pi, \varpi | Пи         |
