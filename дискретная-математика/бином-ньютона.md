# Бином Ньютона

[↑ **Бином Ньютона**](https://ru.wikipedia.org/wiki/Бином_Ньютона) — формула разложения произвольной натуральной степени [двучлена](./../алгебра/алгебраическое-выражение.md#бином) в [многочлен](./../алгебра/алгебраическое-выражение.md#многочлен), расположенный по степеням одного из слагаемых двучлена:

$$
(x + y)^m = x^m + \frac{m}{1!}x^{m-1}y + \frac{m(m-1)}{2!}x^{m-2}y^2 + \cdots + y^m = \sum_{k=0}^m C_m^k x^{m-k}y^k, (\ast)
$$

где $C_m^k =\frac{m!}{k!(m-k)!}$ — [биноминальные коэффициенты](#биноминальный-коэффициент).

Для $n$ слагаемых формула $(\ast)$ принимает вид

$$(x_1 + \dots + x_n)^m = \sum_{k_1 + k_2 + \dots + k_n = m} \frac{m!}{k_1! k_2! \dots k_n!} x_1^{k_1} x_2^{k_2} \dots x_n^{k_n}$$

При произвольном показателе $m$, действительном или даже комплексном, в правой части $(\ast)$ получается, вообще говоря, [↑ биномиальный ряд](https://ru.wikipedia.org/wiki/Биномиальный_ряд).

Постепенное освоение формулы бинома Ньютона, начинавшееся с ее простейших частных случаев (формул «квадрата суммы» и «куба суммы»), прослеживается уже с 11 в. Заслуга [И. Ньютона](./../математики.md#-ньютон-исаак), собственно говоря, состоит в открытии биномиального ряда.
