# Наибольший общий делитель

[↑ **Наибольший общий делитель**](https://ru.wikipedia.org/wiki/Наибольший_общий_делитель) — наибольший из общих делителей целых, в частности натуральных, чисел $а_1, ..., a_n$. Если данные числа не все равны нулю, то такой делитель существует.

Наибольший общий делитель чисел $а_1, ..., а_n$ обычно обозначают символом $(а_1, ..., а_n)$.

Наибольший общий делитель двух натуральных чисел можно найти при помощи [алгоритма Евклида](#алгоритм-евклида). Число шагов, необходимых для отыскания наибольшего общего делителя двух чисел, превосходит не более чем в пять раз число цифр наименьшего из них, записанного в десятичной системе счисления.
